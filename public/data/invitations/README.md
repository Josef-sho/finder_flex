# Invitations Folder

Place invitation files here, named by table number.

## File Naming

Name your invitation files using the table name:
- `Table 1.pdf` or `Table 1.png` for Table 1
- `Table 2.pdf` or `Table 2.jpg` for Table 2
- etc.

Supported formats: `.pdf`, `.png`, `.jpg`, `.jpeg`

The app will automatically find and display the invitation for each table.
